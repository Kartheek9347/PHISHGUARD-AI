About PhishGuard AI

🧠

Project Overview

PhishGuard AI is an intelligent phishing detection system developed as a capstone project. It detects phishing emails and malicious URLs using a hybrid AI approach combining Machine Learning and rule-based analysis.

⚙️

Core AI Model

Uses Logistic Regression trained on TF-IDF vectorized data to classify phishing emails. The model predicts phishing probability based on textual patterns.

📊

NLP & Context Engine

Applies Natural Language Processing to detect urgency, threats, and manipulation patterns commonly found in phishing emails.

🔗

Hybrid Detection System

Combines ML predictions with rule-based scoring to improve accuracy and reduce false positives.

🎯

Objective

Provide a fast, reliable, and explainable phishing detection system for real-world cybersecurity applications.

System Modules

Threat Predictor

ML model that predicts phishing probability using Logistic Regression.

URL Analyzer

Detects suspicious URL patterns (IP usage, long URLs, HTTP).

Context Engine

Analyzes intent like urgency, threats, manipulation.

Feature Intelligence

Extracts features like URL length, symbols, keywords.

Behavioral Engine

Analyzes webpage behavior (redirects, scripts).

Explainable AI

Provides reasons behind predictions.

Database Handler

Stores user data and scan results in SQLite.

Continuous Learning

Retrains model using stored data (future enhancement).
