# 🛡️ PhishGuard AI

PhishGuard AI is an intelligent phishing email detection system developed using Machine Learning, Natural Language Processing (NLP), behavioral analysis, and explainable AI techniques. The system is designed to identify malicious phishing emails, suspicious URLs, and cyber threats with improved accuracy and transparency.

---

# 🚀 Features

- 🔍 Phishing Email Detection
- 🌐 Malicious URL Analysis
- 🧠 Machine Learning-Based Threat Prediction
- 📊 Dynamic Risk Scoring Engine
- 🧾 Explainable AI Predictions
- 📨 Email Context & Metadata Analysis
- ⚡ Behavioral Pattern Detection
- 🔐 Secure User Authentication
- 💻 Flask-Based Web Application

---

# 📌 Core Modules

## 1. Threat Predictor
Uses Logistic Regression with TF-IDF to classify phishing emails.

## 2. URL Analyzer
Detects suspicious URLs, redirects, and malicious patterns.

## 3. Context Engine
Analyzes email metadata and sender-related information.

## 4. Feature Intelligence
Extracts important phishing-related features from emails and URLs.

## 5. Behavioral Engine
Detects abnormal sender behavior and suspicious activity patterns.

## 6. Explainable AI Module
Provides understandable reasoning behind phishing predictions.
